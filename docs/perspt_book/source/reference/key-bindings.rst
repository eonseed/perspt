Key Bindings
============

TUI Chat Mode
-------------

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Key
     - Action
   * - **Enter**
     - Send message
   * - **Ctrl+J**
     - Insert newline in input
   * - **Page Up**
     - Scroll chat up one page
   * - **Page Down**
     - Scroll chat down one page
   * - **Ctrl+Up**
     - Scroll up one line
   * - **Ctrl+Down**
     - Scroll down one line
   * - **Home**
     - Scroll to top
   * - **End**
     - Scroll to bottom
   * - **Ctrl+C**
     - Cancel current stream
   * - **Esc** / **Ctrl+Q**
     - Quit

Review Modal (Agent Mode)
--------------------------

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Key
     - Action
   * - **y**
     - Approve node changes
   * - **n**
     - Reject node changes
   * - **c**
     - Send correction feedback
   * - **e**
     - Open files in editor
   * - **d**
     - Toggle diff view
   * - **q**
     - Cancel review (abort node)

Input Editor
------------

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Key
     - Action
   * - **Left / Right**
     - Move cursor
   * - **Ctrl+A**
     - Move to beginning of line
   * - **Ctrl+E**
     - Move to end of line
   * - **Ctrl+W**
     - Delete word backward
   * - **Ctrl+U**
     - Delete to beginning of line
   * - **Ctrl+K**
     - Delete to end of line
   * - **Backspace**
     - Delete character backward
   * - **Delete**
     - Delete character forward
